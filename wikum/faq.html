Mscgen_js is created to follow the mscgen standard, not the other way 'round. Mscgen was created by and is maintained by Mike McTernan, not by me.

I've found a way mscgen_js can be improved. What do I do?

  • Log an issue in the github repository. You'll need an account on github to do that, however and you might not have one. So you can also try to ...
  • ... contact me directly. I'm on twitter and google plus.
  • Lastly, if you're handy with javascript: create a fork of the source on github, fix it and create a pull request. Be sure to understand that - if I decide to merge it to master - your contribution will be licensed under GNU General Public License v3.
